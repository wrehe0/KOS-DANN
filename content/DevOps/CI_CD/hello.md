## Первый Pipeline на CI/CD в GitHub Actions

### 1. Создайте на **GitHub** новый публичный репозиторий `my-first-cicd`.

В корне репозитория создайте папку `.github/workflows/.`

Внутри создайте файл `hello.yml`. **GitHub** сам предложит вам шаблоны, но для начала скопируйте вот этот простой код, который просто выводит приветствие при каждом пуше.

### 2. Файл `hello.yml`
```yml
name: Мой первый workflow
on: [push] # Запускать при каждом пуше
jobs:
  build:
    runs-on: ubuntu-latest # Использовать последнюю версию Ubuntu
    steps:
      - uses: actions/checkout@v4 # Шаг 1: скачать код из репозитория
      - name: Запустить команду
        run: echo "Привет, мир! Я только что запустил CI/CD!"
```

### 3. Закоммитьте и запушите этот файл в ваш репозиторий

Перейдите на вкладку **Actions** в вашем репозитории на **GitHub**. Вы увидите, как ваш **workflow** запустился, а через минуту загорится зеленая галочка
