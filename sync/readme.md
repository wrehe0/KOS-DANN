# MFUA Git Repository Sync Scripts

## Описание

Скрипты синхронизации Git-репозиториев для **MFUA**  — системы автоматического обновления и синхронизации кодовых баз между двумя репозиториями.

### Назначение

1. Обновить исходный репозиторий командой `git pull`
2. Скопировать всё содержимое (кроме папки `.git`) в целевой репозиторий с заменой существующих файлов
3. Опционально выполнить `git push` в целевом репозитории

### Возможности

- Проверка существования и валидности Git-репозиториев
- Опциональная проверка приватных репозиториев с запросом подтверждения
- Поддержка скрытых файлов (dotfiles)
- Цветной вывод с детальным логированием
- Работа на Linux (Bash) и Windows (PowerShell)
