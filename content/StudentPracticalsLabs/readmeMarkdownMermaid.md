## Самостоятельная работа по созданию проекта c README.md + Mermaid

### Тема - любая техническая или творческая, придумайте сами

### Используемые технологии: Git, Markdown и Mermaid

### Используемые инструменты: Git-Bash, VS Code

Задание:
- Создать новый репозиторий (имя репозитория по-английски, без пробелов и спецсимволов) с файлом `README.md`
- Склонировать новый репозиторий себе на компьютер
- Открыть склонированный репозиторий в **VS Code**
- Модифицировать `README.md`
- `README.md` должен быть отформатировать в **Markdown**, привлекательным, содержать картинку, эмоджи, списки, ссылки, заголовки и т.п. и хотябы одну диаграмму на **Mermaid**

[Пример](https://github.com/lehmanmihail-debug/Static-page)

Советы:

- Для помощи пользуйтесь нейросетью
- Коммиты пишите только по-английски
