# Базовые команды

## Управление Терминалом

Сдвинуть вверх лог выполненных ранее команд

**Ctrl+L**

Очистить экран

```shell
clear
```

Показать истоию команд

```shell
history
```

Выполнить нужную команду из списка **history**

```shell
!62
```

где 62 - номер команды по списку

> Важно! Для переключения межды ранее выполненными командами в терминале можно использовать управляющие клавиши **ВВЕРХ/ВНИЗ**

Ещё раз выполнить последнюю команду

```shell
!!
```

## Файловые операции

Показать текущего каталога

```shell
pwd
```

Показать содержимое текущего каталога

```shell
ls
```

Показать подробное содержимое текущего каталога

```shell
ll
```

Вернуться в домашний каталог пользователя

```shell
cd ~
```

**/** - знак корня каталога

**~** - домашний каталог пользователя
